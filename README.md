# A2UI Vision Generator — GenAI HMI Prototype

> 模拟高端智能座舱设计大脑的交互原型。点击按钮生成随机语音指令，中控屏立即渲染三栏对比决策卡片。

## 预览

用任意浏览器打开 `a2ui-prototype.html` 即可运行，无需构建。

## 原型要素

| 元素 | 说明 |
|------|------|
| **中控屏** | Cyberpunk × Glassmorphism 深色模式车载大屏，含边框、扫描线、状态指示灯、实时时钟 |
| **随机提问按钮** | 霓虹青色胶囊按钮，每次点击随机切换一组语音指令场景（不会重复上一轮） |
| **文案文本框** | 麦克风图标 + 当前语音指令实时显示 |
| **三栏决策卡片** | 每张卡片展示：视觉背景 → 名称 → ⭐评分 → 价格 → 🟢优点标签 → 🔴痛点标签 |

## 数据场景（6 组）

- 约会西餐厅推荐
- 川菜馆推荐
- 日料推荐
- 停车方便火锅店
- 商务宴请粤菜
- 烤肉推荐

## 交互动效

- 卡片错峰 slide-in 入场动画（0 / 80 / 160ms）
- 文案切换 cyan flash 高亮
- 按钮 debounce 防连点
- 始终不回显上一轮场景

## 设计规范

遵循 [A2UI-Vision-Generator-Skill.md](./A2UI-Vision-Generator-Skill.md) 中定义的 Skill 视觉契约与语义色彩映射。

## 技术栈

纯静态 HTML + CSS + Vanilla JS，零依赖。
